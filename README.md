# roll-a-ball-unity
# 🎮 Roll-a-Ball Unity Project

This project is a modified version of the Unity *Roll-a-Ball* tutorial.

## 📌 Description

A simple 3D game where the player controls a ball and collects objects around the map. The project includes custom modifications such as changes in layout, objects, and gameplay elements.

---

## 🚀 How to Run the Game (Windows)

1. Go to the **Releases** section of this repository
2. Download the file `Game.zip`
3. Extract the zip file
4. Open:

   ```
   Game.exe
   ```

---

## 🛠 How to Open the Project in Unity

1. Open Unity Hub
2. Click **"Open project from disk"**
3. Select this project folder
4. Open the project

### ▶️ Run the Scene

1. In Unity, go to:

   ```
   Assets/Scenes
   ```
2. Double click the main scene (e.g. `SampleScene`)
3. Press the **Play (▶️)** button

---

## 🎮 Controls

* Move the ball using:

  * Arrow keys **or**
  * WASD
* Collect all objects to complete the level

---

## ✨ Features / Modifications

* Custom objects and materials
* Modified scene layout
* Added gameplay elements (e.g. obstacles or changes to pickups)

---

## ⚠️ Notes

* The `Library` folder is not included because Unity generates it automatically
* If the project opens with an empty scene, manually open it from:

  ```
  Assets/Scenes
  ```
